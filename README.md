# buildNft.ai App
This is a dapp which using huggingface api to generate a promoted picture and then gets saved on the ipfs network which is then saved on the blockchain.

#Tech Stack 
 1- Solidity 
 2- Hardhat
 3-Ether.js
 4- Spehron
 5- HuggingFace API
 6- NFT.Storage
 7- Nodejs
 8- JavaScript
